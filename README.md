# HospStock - Gestão de Estoque Hospitalar 🏥

Sistema desenvolvido para controle de insumos, medicamentos e materiais hospitalares. Projeto focado em usabilidade e segurança para ambientes de saúde.

## 🚀 Funcionalidades
* **Landing Page:** Apresentação profissional do sistema.
* **Dashboard:** Gráficos interativos com indicadores de estoque crítico e categorias.
* **Gestão de Estoque:** Cadastro de materiais com alerta de estoque mínimo.
* **Movimentações:** Registro de entrada e saída de insumos.
* **Níveis de Acesso:** Diferenciação entre Administrador e Usuário comum.

## 🛠️ Tecnologias Utilizadas
* PHP 8.x
* MySQL (MariaDB)
* Bootstrap 5 (Interface)
* Chart.js (Gráficos)
* Laragon (Ambiente de desenvolvimento)

## 📦 Como instalar o projeto
1. Clone este repositório.
2. Coloque a pasta no seu servidor local (ex: `C:/laragon/www/`).
3. Importe o arquivo `banco_de_dados.sql` no seu gerenciador de banco de dados (HeidiSQL/phpMyAdmin).
4. Certifique-se de que as credenciais no arquivo `config/conexao.php` estão corretas.
5. Acesse `http://estoque.test` (ou o link configurado no seu ambiente).
